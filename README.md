# Task 6: K-Nearest Neighbors (KNN) Classification

## Objective
Understand and implement the K-Nearest Neighbors (KNN) algorithm for classification problems and evaluate its performance.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Iris Dataset

## Steps Performed
1. Loaded the Iris dataset.
2. Displayed the first five rows of the dataset.
3. Checked dataset information and missing values.
4. Split the dataset into training and testing sets.
5. Normalized the features using StandardScaler.
6. Trained the K-Nearest Neighbors (KNN) Classifier.
7. Evaluated the model using:
   - Accuracy
   - Confusion Matrix
   - Classification Report
8. Tested different values of K to compare performance.
9. Visualized the decision boundary.

## Results
- Successfully implemented the KNN classification model.
- Evaluated the model using multiple performance metrics.
- Compared different K values to find a suitable model.
- Visualized the decision boundary for better understanding.
